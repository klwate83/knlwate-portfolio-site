# Keletso Lwate Portfolio Website

A custom VS Code recreation of the one-page portfolio site.

## Files
- `index.html` — page structure and portfolio content
- `styles.css` — responsive dark executive visual design with red accent glow
- `script.js` — mobile navigation, reveal animations, dynamic year, and mailto contact form

## How to run locally
1. Open this folder in VS Code.
2. Install the Live Server extension, or open `index.html` directly in your browser.
3. Replace placeholder contact links in the Contact section.
4. Replace the `KL` profile placeholder with your preferred headshot image when ready.

## Suggested next updates
